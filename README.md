This repository contains the execution code and processed datasets utilized in the paper *Leveraging Large Language Models for Conflict Detection in Software Requirements*.

The code provided here serves as a reference implementation of the experimental pipeline. To keep the repository concise and adaptable, it is intentionally provided as a lightweight framework rather than an end-to-end reproduction package. Running all experiments from the paper requires updating the configuration and implementation details for the specific dataset and language model being evaluated.

The processed datasets included in this repository correspond to those used in our experiments. Users can adapt the provided code to reproduce or extend the experiments with different datasets or models.
